# 🌱 동건루틴

나만의 **할 일 · 루틴 · 목표** 관리 앱. 서버 없이 브라우저(localStorage)에만 데이터를 저장하는 완전 프라이빗 개인 앱입니다.

## 주요 기능

- **오늘** — 날짜·시간별 할 일, 주간 스트립 탐색, **좌우 스와이프로 날짜 이동**, 루틴 체크, 밀린 할 일 관리, 하루 한 줄 회고(기분 이모지)
- **기록** — 체중·칼로리·물(숫자, 누적 더하기 지원) / 독서·운동·식단(텍스트)처럼 매일의 변화를 자유롭게 기록, 목표값·목표 연결, 30일 추이 그래프
- **달력** — 월 달력에서 일정·달성·기분 한눈에 보기, 날짜별 상세
- **루틴** — 요일별 반복, 연속 달성 스트릭 🔥, 보관 기능
- **목표** — 월간 / 분기별 / 연간 목표, D-Day, 마일스톤 체크리스트로 진행률 자동 계산, 할 일 연결
- **할 일 상세** — 서브태스크 체크리스트, 우선순위, 태그, 메모, 검색, 내일로 미루기, 복제
- **통계** — GitHub 스타일 잔디 히트맵, 주간 완료율 추이, 루틴별 달성률·스트릭
- **공유 카드** — 성취율·스트릭·활동 기록을 이미지 카드로 생성해 저장/공유 (데이터는 나가지 않고 카드만!)
- **PWA** — 폰 홈 화면에 설치, 오프라인 동작, 다크모드

## 사용 방법

👉 **https://kingofthestock.github.io/donggun-routine/**

### 폰에 설치하기

- **갤럭시(Android)** — Chrome으로 접속 → 메뉴(⋮) → **홈 화면에 추가** (또는 앱 설치)
- **아이폰(iOS)** — Safari로 접속 → 공유 버튼(□↑) → **홈 화면에 추가**

### 폰 ↔ PC 데이터 옮기기

데이터는 각 기기에만 저장됩니다. 기기 간 이동은:

1. 설정 → **백업 내보내기** (JSON 파일 저장)
2. 파일을 카톡 '나에게 보내기' 등으로 다른 기기에 전달
3. 다른 기기에서 설정 → **백업 가져오기**

> ⚠️ 브라우저 데이터를 지우면 기록도 사라져요. 주기적으로 백업해주세요!

## 프라이버시

- 모든 데이터는 **내 기기 안에만** 저장됩니다 (서버·계정·추적 없음)
- 공유 기능은 통계 이미지 카드만 생성합니다 — 원본 데이터는 어디에도 전송되지 않아요

## 기술

순수 HTML/CSS/JS 단일 파일 + Service Worker(오프라인) + Web App Manifest(설치형 PWA). 외부 라이브러리 0개.

로컬 실행: 폴더에서 `python -m http.server` 후 `http://localhost:8000` 접속.
